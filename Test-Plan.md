# Test Plan

## Scope

Tests cover backend services/controllers/integration foundations, frontend forms/guards/features/PWA, API contract smoke checks, and release readiness.

## Test Types

| Type | Tool | Command |
|---|---|---|
| Backend unit/integration | JUnit, Spring test, Testcontainers where applicable | `cd server && .\gradlew.bat clean build` |
| Backend coverage | JaCoCo | Generated by Gradle build |
| Frontend lint | ESLint | `cd client && npm run lint` |
| Frontend unit/component | Vitest, Testing Library | `cd client && npm test` |
| Frontend coverage | Vitest v8 | `cd client && npm run test:coverage` |
| PWA/responsive | Playwright | `cd client && npm run test:pwa` |
| API smoke | Postman/manual | Postman collection |

## Critical Flow Coverage

- Register/Login/Refresh.
- Admin creates branch/zone/station.
- Kiosk QR -> gamer confirm -> play session.
- Wallet top-up mock, billing, refund.
- Order -> staff -> delivered/cancel.
- LFG -> invitation -> lobby -> chat.
- Device heartbeat/timeout -> alert -> resolve.
- Smart Station command mock MQTT path.
- Notification realtime and read state.
- Dashboard filters and export.

## Exit Criteria

- Backend `clean build` pass.
- Frontend lint/test/build pass.
- No critical security finding open.
- Error log contains only real observed errors.
- Known mocks and limitations are documented.
